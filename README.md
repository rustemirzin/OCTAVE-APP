# O.C.T.A.V.E. — демо мобильного приложения

Интерактивный кликабельный прототип мобильного приложения закрытого сообщества **O.C.T.A.V.E.** —
«цифровой операционной системы сообщества»: нетворкинг, репутация, ИИ-рекомендации знакомств,
форум-группы, мероприятия, безопасные сделки и финансовый кабинет участника.

Тёмная премиальная тема с золотыми акцентами. Без сборки и зависимостей — чистые HTML/CSS/JS,
работает полностью офлайн (аватары, обложки и графики рисуются как SVG).

> 📱 Демо для смартфона. Открывайте на телефоне или сужайте окно браузера до мобильной ширины.

## 📲 Установить как приложение на телефон (PWA)

Демо — полноценное **PWA**: ставится на главный экран и открывается на весь экран без браузерной строки.

**iPhone (Safari):** откройте ссылку → кнопка **«Поделиться»** (квадрат со стрелкой) → **«На экран „Домой"»** → **Добавить**. На рабочем столе появится иконка O.C.T.A.V.E.

**Android (Chrome):** откройте ссылку → меню **⋮** → **«Установить приложение»** / **«Добавить на главный экран»**.

После установки работает и без интернета (файлы кэшируются service worker'ом).

## ▶️ Как открыть

**Вариант 1 — просто файл.** Откройте `index.html` двойным щелчком в браузере.

**Вариант 2 — локальный сервер** (рекомендуется):

```bash
cd octave-mobile-demo
python3 -m http.server 4173   # или: npx serve .
# затем откройте http://localhost:4173
```

## ✨ Что внутри

Реализованы все 12 экранов из макета — плюс десяток дополнительных и живые взаимодействия.

### Главные вкладки (нижняя навигация)
| Вкладка | Что показывает |
|---------|----------------|
| **Главная** | карточка участника с рейтингом, ближайшее мероприятие, ИИ-рекомендации, активный запрос |
| **Люди** | поиск по участникам с живой фильтрацией, фильтры (город, сфера, компетенции), подсказка умного поиска |
| **Сообщения** | чаты / запросы / уведомления; список диалогов с непрочитанными |
| **Сделки** | список сделок с прогрессом + детальная карточка сделки |
| **Профиль** | звание, рейтинг, «о себе», цели, форум-группа, быстрый доступ к разделам |

### Дополнительные экраны
- **Онбординг** — заставка сообщества с входом
- **Рекомендации** — свайп-карточки знакомств (✕ / чат / ♥) с % совпадения и общими целями
- **Профиль участника** — карточка другого человека (познакомиться, написать, в избранное)
- **Мероприятия** — ближайшие / прошедшие + детальная страница события с регистрацией
- **Сделка** — стороны, условия, таймлайн этапов, чат по сделке, арбитраж
- **Финансы** — баланс пая, начисленная прибыль, взносы, доход, вознаграждения 1 и 2 уровня
- **Репутация** — trust score, динамика рейтинга (график), 5 составляющих
- **Достижения** — бейджи участника
- **Форум-группы** — своя «восьмёрка» (капитан/наставник, встречи) и другие группы
- **Чат** — переписка с отправкой сообщений и авто-ответом
- **Меню** — запросы, цели, мероприятия, избранное, приглашения, обучение, база знаний, настройки
- **Реферальная ссылка**, **Обучение / база знаний**, **Настройки приватности (2FA, видимость)**, **Уведомления**, **Запросы**, **Цели**

### Что кликается
Переключение вкладок, переходы по карточкам, кнопки «назад», свайп рекомендаций, живой поиск,
отправка сообщений в чате, нижние шторки (новый запрос, фильтры), копирование реф-ссылки,
тосты-подтверждения действий, переключатели в настройках.

## 🛠 Кабинет администрации (для управляющих сообщества)

Отдельный **адаптивный дашборд** руководителя — `admin.html` (откроется и на компьютере, и на телефоне).
Вход: на широком экране — кнопка над телефоном; на телефоне — меню приложения → «Кабинет администрации».

Разделы (по ТЗ, раздел «Административная панель»):
- **Обзор** — KPI (участники, выручка, подписки, trust score, сделки, города), график роста к цели 1 000 000, выручка, **воронка вступления**, распределение по званиям, зоны внимания, лента действий.
- **Участники** — единая база: звание, роль, репутация, статус подписки, фильтры.
- **Кандидаты** — канбан-воронка: Бриф → Поручительство → Собеседование → Проверка СБ → Присяга → Активация.
- **Репутация и жалобы** — trust score, жалобы, **арбитраж**, репутационные риски.
- **Безопасность · СБ** — проверки кандидатов, 2FA, антифрод, RBAC/ABAC, аудит.
- **Сделки (escrow)** — гарант-расчёты, комиссия, статусы, споры.
- **Финансы** — выручка, MRR, структура доходов, выплаты, модель пая.
- **Форум-группы**, **Мероприятия** (регистрации, QR check-in), **Обучение**.
- **География** — страны, города, сенаторы, отделения.
- **Роли и доступы** — матрица RBAC и иерархия сообщества.
- **Журнал действий** — неизменяемый аудит-лог.

## 🗂 Структура

| Файл | Назначение |
|------|-----------|
| `index.html` | каркас: рамка телефона, статус-бар, контейнеры экранов и нижней навигации |
| `styles.css` | дизайн-система: тёмная тема + золото, все компоненты |
| `app.js` | данные (`ME`, `PEOPLE`, `EVENTS`, `DEALS`, …), SVG-генераторы, экраны, роутер, взаимодействия |
| `admin.html` / `admin.css` / `admin.js` | кабинет администрации: дашборд, графики, таблицы, воронка, RBAC |

Чтобы поменять контент — редактируйте объекты данных в начале `app.js`.
